# ProjectTrace 文档总览

## 1. 项目简介

ProjectTrace 是一个面向公司内部的项目跟踪、过程管理、OKR 对齐和信息透明化平台。  
系统规划采用前后端分离架构：

- 前端：Vue3
- 后端：Python + FastAPI
- 开发阶段关系数据库：SQLite
- 开发阶段文档/扩展数据存储：文件型数据库
- 缓存与消息辅助：Redis（可选）
- 部署方式：Docker Compose

## 5. 说明

关于开发阶段存储方案：

- 当前开发阶段采用更轻量的方案：
  - 结构化数据使用 SQLite
  - 文档型/扩展型数据使用文件型数据库
  - Redis 保持可选
- 后续进入多人协作或正式部署阶段时，可再切换到 PostgreSQL 和 MongoDB
