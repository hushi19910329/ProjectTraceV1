# ProjectTrace 文档总览

## 1. 项目简介

ProjectTrace 是一个面向公司内部的项目跟踪、过程管理、OKR 对齐和信息透明化平台。  
系统采用前后端分离架构：

- 前端：Vue3
- 后端：Python + FastAPI
- 关系数据库：PostgreSQL（主）
- 开发阶段文档/扩展数据存储：文件型数据库
- 缓存与消息辅助：Redis（可选）
- 对象存储：MinIO（已接入本地部署）
- 部署方式：Docker Compose

关于当前存储方案：

- 当前采用：
  - 结构化数据使用 PostgreSQL

- 本地基础设施已支持 PostgreSQL、MongoDB、MinIO，可用于联调和后续迁移验证
