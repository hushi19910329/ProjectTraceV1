# ProjectTrace 文档总览

## 1. 项目简介

ProjectTrace 是一个面向公司内部的项目跟踪、过程管理、OKR 对齐和信息透明化平台。  
系统规划采用前后端分离架构：

- 前端：Vue3
- 后端：Python + FastAPI
- 关系数据库：PostgreSQL
- 文档/扩展数据存储：MongoDB
- 缓存与消息辅助：Redis
- 部署方式：Docker Compose

## 5. 说明

关于 MongoDB：

- 严格来说，MongoDB 是文档数据库，不是传统意义上的图数据库
- 当前按项目要求，将 MongoDB 用于承载文档型、附件型、扩展结构、动态配置、复杂嵌套信息等数据
- 如果后期确实需要更强的图关系查询能力，可再评估 Neo4j 等真正的图数据库
